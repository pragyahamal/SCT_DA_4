# Marketing Campaign EDA

## Objective

To perform Exploratory Data Analysis (EDA) on a marketing campaign dataset and generate business insights based on customer behavior and campaign performance.

## Features

* Data cleaning and preprocessing
* Missing value and duplicate checking
* Income distribution analysis
* Product spending analysis
* Correlation heatmap visualization
* Campaign response analysis
* ROI-based business recommendations

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab
* GitHub

## Outcome

The analysis helped identify customer spending patterns and provided recommendations for improving marketing strategies and budget allocation based on ROI.
